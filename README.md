# eat-scan
爸媽簡易說明書（可直接複製）

我們家小朋友常常會出現這種狀況：明明吃沒幾口就說「我飽了」，或是吃到一半開始分心、拖時間，結果主食沒吃完、營養不夠。為了讓「再多吃一點點」變得更有趣、更容易配合，我們做了這個 《吃飽沒測試掃描器》——它是一個「假裝很科學」的掃描器，用 X 光風格介面顯示飽足百分比，讓孩子覺得被測量、被判定，就比較願意再吃幾口（純家庭娛樂用）。

使用方式（演就對了）

打開手機手電筒（更像真的儀器）。

打開 APP，照畫面提示：
把食指放在任一顆鏡頭前方（其實不用真的對很準）。

按下底部大按鈕「開始掃描」或直接按按鈕上的指定區域。

等掃描結束（會有進度條、嗶一聲、輕微震動），畫面會顯示：

飽足程度 %

建議：還可以再吃幾口 / 已吃飽

按鈕區域說明（從左到右）

底部那顆大按鈕其實分成 4 段（看起來一顆，實際上是不同結果）：

最左 20%：顯示「幾乎沒吃 / 還可以吃很多口」

中間 40%（最常按這段）：顯示「還可以吃很多」，通常會說 再吃 8–12 口

再右邊 30%：顯示「快吃飽」，通常會說 再吃 2–4 口

最右 10%（很難按到）：顯示「已吃飽 / 停止進食」

✅ 想讓孩子再多吃一些：就按 中間那一大段（40%）。

爸媽一句話演技（可照念）

「我掃一下確認喔～」

「你看！還有空間耶！」

「好，照系統說的，再吃 8～12 口就收工！」

快吃飽時：「剩 2～4 口就好，慢慢吃～」
